端口: 5502
    说明: 标志位置检测服务
    请求参数:
        Headers:
            Content-Type: application/json (optional)
        files:
            file: 二进制图片数据
    响应数据:
        "code": 状态码，0表示成功，其他表示各种错误
        "msg": 消息
        "serverTime": 服务器响应时间
        "data": 检测框列表
******************************************************
端口: 5503
    说明: 标志分割服务
    请求参数:
        Headers:
            Content-Type: application/json (optional)
        files:
            file: 二进制图片数据
    响应数据:
        "code": 状态码，0表示成功，其他表示各种错误
        "msg": 消息
        "serverTime": 服务器响应时间
        "data": 图片mask
